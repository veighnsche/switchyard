# Security Policy

## Supported Versions

We currently support the latest published version of Switchyard.

## Reporting a Vulnerability

Please report security issues privately. Open a minimal GitHub issue referencing a security concern and request a maintainer contact, or email the maintainer listed on the repository.

Include:
- Affected version(s) and environment
- Reproduction steps or PoC
- Impact assessment (what could an attacker achieve)

We will acknowledge receipt within 72 hours and work on a fix or mitigation. Coordinated disclosure is appreciated.
